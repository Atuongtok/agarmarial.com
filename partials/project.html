AI Resume Analyzer

Our intelligent system helps HR teams screen, analyze, and match resumes with job requirements using advanced AI and NLP techniques.

Key Features

  • ✔ Resume Parsing (PDF, DOCX, Text)
  • ✔ Skill Extraction using NLP
  • ✔ AI-based Resume Classification
  • ✔ Job Matching with Similarity Scoring
  • ✔ Interactive Dashboard for Recruiters

How It Works

Upload your resume ➜ The system extracts & analyzes your data ➜ Matches your profile with the best-fit jobs using AI ➜ Displays rankings and suggestions for recruiters.

Technology Stack

Frontend: HTML, CSS, JavaScript
Backend: Python, Flask
AI Models: NLP, CNN-based Classifier, Resume Matching with Cosine Similarity
Database: SQLite/MySQL
